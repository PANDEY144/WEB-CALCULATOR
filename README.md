🧮 Web Calculator
📌 Overview

This project is a responsive Web Calculator built using HTML, CSS, and JavaScript.
It performs basic arithmetic operations with a clean, modern, and user-friendly interface.

The main goal of this project is to strengthen core frontend skills such as DOM manipulation, event handling, and responsive UI design.

📊 Features
➕ Perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
🧠 Real-time calculation display
🧹 Clear/reset functionality
📱 Fully responsive design for all devices
🎯 Simple and intuitive user interface
🚀 Key Highlights
Designed a clean and minimal UI for better user experience
Implemented interactive buttons using JavaScript events
Ensured responsive layout using CSS
Focused on usability and smooth performance
📈 Functionalities
Input numbers using on-screen buttons
Perform calculations instantly
Display results dynamically
Handle invalid inputs gracefully
🛠️ Tech Stack
HTML – Structure of the calculator
CSS – Styling and responsive layout 🎨
JavaScript – Logic and functionality ⚡
🏁 Conclusion

This project demonstrates:
✔️ Strong understanding of JavaScript logic
✔️ Efficient DOM manipulation
✔️ Ability to create responsive and user-friendly interfaces
✔️ Practical implementation of frontend development concepts
